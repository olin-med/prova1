# Prova 1 Ólin

## Descrição

Este repositório contém o código relacionado à uma aplicacão web capaz de salvar posicões de um robô.

## Uso

1. **Instalação de Dependências**: Certifique-se de ter todas as dependências instaladas. Para tal, execute o comando `pip install -r requirements. txt`
3. **Execução do Programa**: Execute o programa principal (Execute o mesmo através de python3/python src/app.py).
